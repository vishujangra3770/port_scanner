# 🔍 Port Scanner

A Python-based tool to scan open ports on a target system using socket programming, multithreading and identify potential security risks.

## 🛠 Technologies Used
- Python
- Socket Programming
- Multithreading
  
## Features
- Scan common ports
- Detect open services
- Fast scanning with threading

![Demo](port-scanner.gif)

## 🚀 Usage
1. Clone the repository:
```bash
git clone https://github.com/vishujangra3770/port-scanner-python.git
cd port-scanner-python
```
2. Run the script:
```bash
python port_scanner.py
```
3.Enter the target IP or domain when prompted (Example).
```bash
scanme.nmap.org
```
